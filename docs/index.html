Material Biases as Compositional Resources in Text-to-Audio Models

Abstract Audit Design Interactive Audit Workflow & Logs Reproduce

Abstract

Text-to-audio models mediate musical agency through training data, annotations, text encoders, and diffusion controls. This practice-based paper presents a workflow which treats those mediations as compositional material. Using Stable Audio Open, we introduce two heuristics for exposing the model's learned associations: shallow prompting, which uses abstract or underspecified prompts to probe grounding strength in learned audio--text associations, and partial denoising, which truncates the sampler schedule to keep noise prior and text conditioning unresolved. A model-specific CLAP audit triangulates listening observations across prompt families and denoising depth. The exposed material is then re-appropriated through concatenative synthesis in a live performance. The paper contributes a practice-led method for making model-specific text-to-audio biases audible and reusable in performance.

Audit Design

The companion page centers on a seed-matched audit of Stable Audio Open. The audit compares prompt specificity and denoising depth under a consistent generation setup, making it possible to listen across controlled prompt families and sampler trajectories.

Workflow & Generation Logs

  • ComfyUI workflow (JSON) — importable graph used for the audit settings.
  • Generation log (CSV) — full record of runs (prompt, seed, sampler, scheduler, CFG, denoise, steps, duration, notes).
  • Audit manifest (CSV) — 360 matched render jobs with prompt family, prompt slot, seed slot, CFG, sampler, scheduler, denoise, and audio paths.
  • Rendered audio metadata (CSV) — extracted metadata used to verify prompt, seed, CFG, sampler, scheduler, steps, denoise, and duration.
  • Metric tables: family separation, distance to unconditioned, within-family dispersion, exact-prompt similarity, prompt-text similarity matrix, and prompt confusion.
  • Figures: separation, distance, dispersion, exact-prompt similarity, and joint PCA trajectory.

Reproducing the Audit

To regenerate an audit clip locally:

1. Install ComfyUI and the Stable Audio Open model weights from HuggingFace.
2. Load workflow/stable_audio_open.json in ComfyUI.
3. Set the prompt, seed, CFG, steps, sampler, scheduler, and denoise as listed in audit/manifest.csv.

The generation log and rendered metadata provide a second check on the intended settings for each published clip.
